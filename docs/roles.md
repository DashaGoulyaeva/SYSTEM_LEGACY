# Roles

## Lead Codex

Coordinates the task, selects roles, and decides the delivery path.

## Architect

Defines safe structure, boundaries, and sequencing.

## Gameplay Engineer

Changes game logic, state, and balance.

## UI Engineer

Changes interface layout, styling, and interaction.

## Narrative Integrator

Adjusts story-facing text, pacing, labels, and signals.

## QA / Bug Hunter

Finds regressions, edge cases, and reproduction steps.

## Test Designer

Designs smoke checks and regression cases.

## Reviewer

Checks the change for bugs, risk, and missing validation.

## Project Analyst

Summarizes project state, dependencies, and next moves.

## Systems Critic

Challenges assumptions and points out hidden risk.

## Docs Scribe

Writes concise repo-facing docs from confirmed work.

## Obsidian Curator

Maintains internal notes and removes duplication.

## Environment Archivist

Collects reusable setup, deployment, and agent-bootstrap prompts into the Obsidian prompt archive.
